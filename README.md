# The Collective Counsel (TCC)

A modern, responsive frontend website for The Collective Counsel - a professional counseling services organization.

## Features

- **Modern Design**: Clean, professional design with contemporary color scheme and typography
- **Responsive Layout**: Fully responsive design that works on mobile, tablet, and desktop devices
- **Accessible**: Built with accessibility in mind (WCAG compliant)
- **Interactive**: Smooth scrolling navigation, form validation, and scroll animations
- **Fast Loading**: No heavy frameworks, optimized for performance

## Sections

- **Hero**: Welcoming landing section with call-to-action
- **Services**: Comprehensive overview of counseling services offered
- **About**: Information about the organization and its values
- **Team**: Meet the expert counselors
- **Testimonials**: Client success stories
- **Contact**: Contact form and office information
- **Footer**: Quick links, social media, and legal information

## Project Structure

```
TCC/

├── css/
```

## Technologies Used

- **HTML5**: Semantic markup for better accessibility and SEO
- **CSS3**: Custom properties, Flexbox, Grid, and modern styling
- **JavaScript (ES6+)**: Vanilla JS for interactivity
- **Google Fonts**: Inter and Playfair Display typography

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. No build process or dependencies required!
